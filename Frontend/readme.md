# Chat Nova - Decentralized Messaging Frontend

A next-generation futuristic, blockchain-themed React.js frontend for Chat Nova - a decentralized anonymous messaging platform with content-level threat detection.

## ✨ Futuristic Features

### 🎨 Ultra-Modern UI Effects
- **Holographic Elements**: Shimmering holographic overlays on cards and forms
- **Neon Glow Effects**: Pulsating neon borders and text shadows
- **Particle System**: Animated particle background with connecting lines
- **Glassmorphism**: Advanced frosted glass effects with backdrop blur
- **Cyberpunk Grid**: Animated scanline and grid overlays
- **Glitch Effects**: Subtle glitch animations on title text

### 🌟 Advanced Animations
- **Smooth Transitions**: Cubic-bezier easing for natural motion
- **Glow Pulsing**: Dynamic glow effects on interactive elements
- **Floating Particles**: Ambient particle animations throughout
- **Shimmer Effects**: Gradient shimmer on buttons and inputs
- **Status Indicators**: Animated online/offline status dots
- **Message Slides**: Smooth message entry animations

### 💎 Interactive Elements
- **Custom Cursor**: Neon-themed cursor with glow effect
- **Hover Transformations**: Scale and glow on hover
- **Focus Glows**: Enhanced input focus with multi-layer shadows
- **Button Ripples**: Futuristic button press effects
- **Card Elevations**: 3D-style card hover effects

### 🔮 Visual Enhancements
- **Gradient Backgrounds**: Multi-layer radial gradients
- **Border Animations**: Animated glowing borders
- **Text Shadows**: Multi-layer neon text effects
- **Icon Animations**: Rotating and pulsing icons
- **Scrollbar Styling**: Custom neon-themed scrollbars

## 🚀 Core Features

- Anonymous user identity generation with blockchain-style keys
- Real-time messaging interface with encryption simulation
- Government monitoring portal with AI threat detection
- Contact management with online/offline status
- Message blockchain hash tracking
- Privacy-first design with anonymous identities

## Installation

```bash
npm install
```

## Run Development Server

```bash
npm start
```

The app will open at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Tech Stack

- React.js 18
- CSS3 with advanced animations
- Canvas API for particle effects
- Modern ES6+ JavaScript
- Glassmorphism & Neumorphism design
